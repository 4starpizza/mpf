mpf.system.window module

Window manager which controls any pop up windows from MPF. Used to display game information, status, tests, keyboard-to-switch mapping, on screen DMD, etc.

class mpf.system.window.WindowManager(machine)

Bases: mpf.system.display.MPFDisplay

Parent class for the Pygame-based on screen Window Manager in MPF.

There is only one Window Manager per machine. It’s used for lots of things, including displaying information about the game, an on-screen DMD, and for capturing key events which are translated to switches.

register_handler(event, handler)

Registers a method to be a handler for a certain type of Pygame event.

Parameters:
  • event – A string of the Pygame event name you’re registering this
  • handler – A method that will be called when this Pygame event is
update()

Called from a timer based on this display’s fps settings. Updates the display and collects key and mouse events.
