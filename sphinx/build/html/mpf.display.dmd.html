mpf.display.dmd module

MPF display plugin the dot matrix display, used for both physical and on- screen DMDs.

class mpf.display.dmd.DMD(machine)

Bases: object

Base class for a traditional dot matrix display (DMD) in a pinball machine. This class is used to control a physical DMD connected via the 14-pin header on the controller, and also to create an on-screen representation of that DMD in a Window.

Note that if you want to control a “color DMD”, that’s actually and LCD screen which means you’d use the LCD class (with a filter to make it look like dots), not this DMD class.

Parameters:machine – A reference to the main machine controller object.

update_physical(surface)

Updates the physical DMD with the Pygame surface that is passed to it.

Parameters:surface – A Pygame surface that will be written to the physical DMD.

Note: The Pygame surface passed must have a color depth of 8-bits, and the physical DMD will only read integer values 0 through the bit depth of the DMD, typically 4 (meaning it only looks for values 0-3) or 16 (meaning it looks for 0-15). Any value higher than that will be rendered as full brightness.

update_screen()

Updates the on screen representation of the physical DMD. This method automatically scales the surface as needed.
