mpf.system.display module

A generic display interface which is used for all types of displays, including DMD, LCD, alpha, segment, console, etc..

class mpf.system.display.DisplayController(machine)

Bases: object

Parent class for the Display Controller in MPF. There is only one of these per machine. It’s responsible for interacting with the display, regardless of what type it is (DMD, LCD, alphanumeric, etc.).
