Adds a Pygame font object to the font cache.

  • font_name – String of what you want to refer to this font as.

get_font(font=None, size=None, cache=True)

Returns a Pygame font rendering object for a given font name and size. Checks cache first and then falls back to disk. Optionally caches new font objects for reuse.

Parameters:
  • name – String name of the font requested.
  • size – Integer of the MPF size of the font you want.
  • cache – Boolean as to whether the ront rendering object that’s created should be cached.

Returns: A Pygame font rendering object.

locate_font_file(file_name)

Takes a file name and tries to find the font file on disk. First it checks the machine_folder fonts folder to see if there’s a machine- specific font, then it tries the MPF system fonts folder.

Parameters:file_name – String of the font file name

Returns: String of the full path (path + file name) of the font.
